## Publish
1. **Проверить версию пакета**
2. **Собрать пакет** `npm run build` 
3. Запаковать 'npm pack`
4. Удалить артефакты разработки `html360 uninstall-menu` `npm unlink` `npm uninstall -g html360`    
5. Установить глобально из архива `npm install -g ./html360-x.x.x.tgz`
6. **Проверить все команды html360**
7. **Проверить CHANGELOG**
8. Создать коммит в мастер 
10 **Добавить tag** `git tag -a vX.X.X -m "Release version x.x.x"`
11. **Последний раз все проверить**
12. Логин `npm login`
13. Выложить `npm publish`
14. Создать релиз на github
